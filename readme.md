## 项目准备

* 配置 `node`

 ```
 curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.30.2/install.sh | bash
 nvm install v4.5.0
 nvm alias default v4.5.0
 ```
* 安装依赖

 ```
 npm install
 ```

## 相关指令

### 运行服务器
```
npm start
```

### 构建
```
npm build
```
